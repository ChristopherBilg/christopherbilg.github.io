JS Temporal

The Temporal API is a modern replacement for JavaScript's legacy Date object. It provides immutable, well-typed objects for instants, plain dates, plain times, zoned date-times, durations, and arithmetic across calendars and timezones — without the footguns of Date's mutable, locale-leaky behavior.
